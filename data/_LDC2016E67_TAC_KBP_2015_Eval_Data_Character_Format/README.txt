Since this year we decided to change our scoring method to character based methods. We hereby release the gold standard
files of TAC KBP 2015 in character format for convenient evaluation.

TAC_KBP_2015_EvalEventHopper_char.tbf is converted from EvalEventNugget20150903.tbf. This file appears in LDC2015E95 and
LDC2015R26, which contains event nugget annotation only.

TAC_KBP_2015_EvalEventNugget_char.tbf is converted from EvalEventHopper20150903.tbf. This file appears in LDC2015R26,
which contains both event nugget and hopper annotation.
